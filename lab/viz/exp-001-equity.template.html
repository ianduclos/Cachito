Equity curves — P(win) by own dice

Calzo repricing — breakeven P(exact) by state

Calzo pays +1 die if exact, −2 if not. Pricing that trade on the equity curve gives the breakeven probability at which calling is worth it. The live bot uses a flat 0.72 — too loose with 4 dice, far too tight with 1 die (you can only lose one). At 5 dice the reward is capped away entirely: value-Calzo is never correct.

Starter effect — within identical stack states

Comparing starters vs non-starters inside the same stack configuration (1,240 states with n≥200 on both sides). Starting is a small consistent edge — except where it isn't: with full hands heads-up, opening leaks information; with one die each, opening is nearly winning.

—

Read with care. Equity here is measured under the six-policy heuristic baseline population, not game-theoretic value — it will shift as the population gets stronger. Observations within one game share a winner, so effective sample sizes are smaller than the raw n. Calzo breakevens use table-size marginals (not full state conditioning) and ignore the round-starting initiative a successful caller gains, which nudges true thresholds slightly lower.
